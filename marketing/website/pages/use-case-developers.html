Vibe Coding

Голосовые заметки в текст для разработчиков

Транскрибация голосовых заметок в текст для AI-кодинга. Наговорите идею -- получите текст для промпта в Cursor, Claude или Copilot. Бесплатно, оффлайн.

Знакомые проблемы?

Идеи теряются

Идеи приходят во время прогулки, пробежки или перед сном. Нужно быстро наговорить и потом получить текст.

Нужен интернет

Облачные инструменты не работают без Wi-Fi. В самолете, на даче или в кафе с плохим сигналом -- никакой расшифровки.

Подписки надоели

MacWhisper стоит $29, SuperWhisper -- $10/мес. Разработчики-одиночки и так тратят на инструменты. Ещё одна подписка -- перебор.

Как Traart решает эти задачи

Нативное macOS-приложение

Drag & drop аудиофайла, результат через минуты. Никакой командной строки или Python-зависимостей.

Работает без Wi-Fi

В самолете, кафе без Wi-Fi, на даче. Один раз скачал -- пользуешься вечно. GigaAM v3 в 2x точнее Whisper на русском.

Бесплатно навсегда

Нет подписки, нет пробного периода. Скачал -- работает. Оптимизировано для Apple Silicon с MPS-ускорением.

Частые вопросы

Как использовать Traart для vibe coding?

Запишите голосовую заметку с описанием фичи или бага, расшифруйте в Traart, скопируйте текст в промпт Cursor, Claude Code или Copilot. Быстрее и точнее, чем печатать.

Traart работает без интернета?

Да, полностью оффлайн. Модель GigaAM v3 загружена локально на Mac. Расшифровка не требует подключения к сети -- работает в самолете, в метро, без Wi-Fi.

Можно настроить авто-мониторинг папки?

Да. Укажите папку -- Traart автоматически расшифрует каждый новый аудиофайл. Удобно для записей с Zoom-стендапов или голосовых заметок, которые попадают через AirDrop.

В какие форматы можно экспортировать?

Markdown, TXT и JSON с таймкодами. Markdown удобен для документации, JSON -- для автоматизации через скрипты и пайплайны.
