Детальный анализ конкурентов

M

MacWhisper

macOS EUR 29-59

Десктопная оболочка для Whisper large-v3 от OpenAI

Преимущества

  • +Мультиязычная поддержка (99+ языков)
  • +Работает локально на Mac
  • +Удобный нативный интерфейс
  • +Экспорт в SRT/VTT для субтитров

Недостатки vs Traart

  • -WER ~16% на русском vs 8.3% у Traart (в 2x хуже)
  • -Платный: EUR 29 (Basic) / EUR 59 (Pro+)
  • -Диаризация только в Pro+ (EUR 59)
  • -Нет авто-мониторинга папок

Вывод: MacWhisper — хорошее приложение для мультиязычных задач, но для русской речи модель Whisper large-v3 дает WER ~16% vs 8.3% у GigaAM v3. Traart бесплатен и включает диаризацию.

Y

YandexSpeechKit

Облако ~$0.32/час

Облачный STT от Яндекса, серверы в РФ

Преимущества

  • +Хорошее качество русской речи (WER ~8-12%)
  • +Потоковое распознавание в реальном времени
  • +API для интеграции в приложения
  • +Серверы в России (152-ФЗ)

Недостатки vs Traart

  • -Платный: ~$0.32/час (~$19 за 60 часов)
  • -Аудио отправляется в облако Яндекса
  • -Требует регистрации, API-ключа, Yandex.Cloud
  • -Нет десктопного приложения — только API

Вывод: Ближайший по качеству конкурент, но требует оплаты и отправки аудио в облако. Для индивидуального использования Traart предоставляет сопоставимое качество бесплатно и с полной приватностью.

G

Google Speech-to-Text

Облако ~$0.96/час

Облачный STT от Google, модели Chirp 2 / USM

Преимущества

  • +125+ языков с Chirp 2
  • +Диаризация и автопунктуация
  • +Enterprise масштабируемость
  • +Интеграция с Google Cloud

Недостатки vs Traart

  • -WER ~16.7% на русском vs 8.3% у Traart
  • -Дорого: ~$0.96/час ($57.6 за 60 часов)
  • -Данные обрабатываются серверами в США
  • -Только API — нет десктопного приложения

Вывод: Для мультиязычных enterprise-задач Google STT хорош, но для русской речи уступает GigaAM v3 вдвое. Самый дорогой вариант в сравнении — $0.96/час.

T

TurboScribe

Web $10-20/мес

Веб-сервис на базе Whisper с дополнительной обработкой

Преимущества

  • +Удобный веб-интерфейс
  • +Диаризация и субтитры
  • +Экспорт в PDF, DOCX, SRT
  • +Бесплатный план (3 файла/день)

Недостатки vs Traart

  • -WER ~16% на русском (базируется на Whisper)
  • -Подписка $10-20/мес ($120-240/год)
  • -Аудио загружается на серверы
  • -Лимиты на бесплатном плане

Вывод: TurboScribe удобен как веб-сервис, но использует ту же модель Whisper с WER ~16%. За $10-20/мес вы получаете качество в 2x хуже бесплатного Traart.

N

Notta

Web / iOS $13-59/мес

Облачный сервис с фокусом на заметки из встреч

Преимущества

  • +Интеграция с Zoom, Meet, Teams
  • +AI-суммаризация встреч
  • +Мобильное приложение
  • +Реальное время + загрузка файлов

Недостатки vs Traart

  • -Среднее качество русской речи (~15% WER)
  • -Дорогая подписка: $13/мес (Pro) / $59/мес (Business)
  • -Все данные в облаке
  • -Нет работы оффлайн

Вывод: Notta удобна для командной работы с встречами, но за $13-59/мес вы получаете среднее качество русской речи и зависимость от облака.

O

Otter.ai

Не поддерживает русский язык. Работает только с английским. Не является конкурентом для русскоязычной транскрибации.

Traart побеждает в каждой категории

Качество (русский)

SOTA

WER 8.3% — лучший результат среди всех моделей. В 2x точнее Whisper (~16%), точнее Google (16.7%).

Цена

Бесплатно

$0 навсегда. MacWhisper стоит EUR 29-59, TurboScribe $120-240/год, Google $57.6 за 60 часов обработки.

Приватность

100% локально

Ни байта в облако. MacWhisper тоже локальный, но остальные 4 конкурента отправляют аудио на серверы.

Диаризация

Бесплатно

Встроенная диаризация через pyannote без дополнительной платы. MacWhisper берет EUR 59 за Pro+ с диаризацией.

Автоматизация

Уникально

Единственный продукт с авто-мониторингом папок. Ни один конкурент не предлагает эту функцию.

Простота

Menu bar

GUI в menu bar без CLI. Yandex и Google — только API. Установка: перетащить в Applications.

Полностью бесплатен: Конкуренты берут от EUR 29 до $59/мес. С Traart вы платите $0 за лучший результат.

Максимальная приватность: 100% локальная обработка на Apple Silicon. Ни один байт аудио не покидает ваш Mac.
