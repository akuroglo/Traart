Транскрибация для юристов

Транскрибация для юристов с сохранением адвокатской тайны. 100% оффлайн расшифровка записей -- ни один байт данных клиентов не покидает ваш Mac. Бесплатно.

Конфиденциальность на уровне архитектуры

Не обещания -- технические гарантии

Zero Data Transmission

В коде приложения нет сетевых вызовов. Нет HTTP-клиентов, нет WebSocket, нет телеметрии. Аудио обрабатывается локальной моделью GigaAM v3.

Без аккаунта

Нет регистрации, нет логина, нет лицензионного ключа. Traart не знает, кто вы. Аудит невозможен по определению.

152-ФЗ по архитектуре

Персональные данные не передаются третьим лицам, потому что приложение физически не может этого сделать. Соответствие 152-ФЗ гарантировано дизайном.

Проблемы облачной транскрибации для юриста

Данные на чужих серверах

Облачные сервисы (Google, Yandex, Notta) передают аудиозаписи на серверы третьих лиц. Запись беседы адвоката с клиентом -- на сервере в облаке.

Нарушение адвокатской тайны

Передача записи беседы с клиентом третьим лицам может нарушить ст. 8 ФЗ "Об адвокатской деятельности". Облачный сервис -- это третье лицо.

Стоимость сервисов

Профессиональные транскрибации: Trint $52-100/мес, ручная расшифровка $1-2/мин. На объёмах юридической практики расходы ощутимы.

Облачный сервис vs Traart

Облачный сервис

  • Аудио передаётся на серверы
  • Требуется аккаунт с персональными данными
  • Данные хранятся на серверах провайдера
  • Риск утечки при взломе сервиса
  • Нет контроля над обработкой данных

Traart

  • Всё обрабатывается локально на Mac
  • Без аккаунта, без регистрации
  • Файлы остаются на диске пользователя
  • Нет серверов -- нечего взламывать
  • Open-source: можно провести аудит кода

Как начать работу

1

Установите Traart

Скачайте .dmg, перетащите в Applications. Без регистрации, без аккаунта. Модели загрузятся автоматически при первом запуске.

2

Поместите запись в папку

Скопируйте аудиозапись в настроенную папку мониторинга. Traart поддерживает MP3, M4A, WAV, MP4 и другие форматы.

3

Получите расшифровку

Текст с таймкодами и разделением спикеров появится рядом с аудиофайлом. Всё обработано локально -- проверяемо, безопасно.
