Медицинские данные -- специальная категория. Обработка требует письменного согласия. Передача в облако создаёт риски нарушения.

Как начать

1

Установите на рабочий Mac

Скачайте .dmg, перетащите в Applications. Без регистрации, без ввода данных клиники. Модели загрузятся при первом запуске.

2

Настройте папку для записей

Укажите папку, куда сохраняются аудиозаписи приёмов. Traart будет автоматически обрабатывать каждый новый файл.

3

Используйте расшифровку

Текст с разделением по спикерам и таймкодами -- основа для документации. Всё обработано локально на вашем Mac.
